Comment utilisé du css :

créer un fichier css Dans le fichier html écrire link href="[lien de la page css]" rel="stylesheet"
Dans le fichier css écrire: selector (ce que l’on sélectionne dans l’HTML (exemple: p)) {
property:(ce que l’on veut modifier (exemple: color)) value; (la valeur que l’on applique (exemple red) }
